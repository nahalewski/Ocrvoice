Cross-Server Sync

Compares this server's Movies and TV Shows against a second Jellyfin server using Jellyfin metadata (provider IDs and season/episode numbers), not file names, and fills in what is missing without overwriting anything.

This plugin can only write files to this server's disks, so transfers always pull here. To fill gaps on the other server, install this plugin there too and point it back at this one.
